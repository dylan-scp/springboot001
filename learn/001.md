### Thứ tự ưu tiên của file config: application.properties

#### 1. File nằm trong thư mục config của thư mục gốc- trong này là demo/config/application.properties
#### 2. File nằm trong thư mục gốc - trong này là demo/application.properties
#### 3. File nằm trong thư mục config của src - trong này là demo/src/resources/config/application.properties
#### 4. File nằm trong thư mục resource - trong này là demo/src/resource/application.properties

## File application.properties sẽ chỉ nằm trong vị trí trên.
## Luôn lưu tiên lấy giá trị trong file config application.properties trước file application.yml
